# Smart Guardian System 🚨

An AI-powered Smart Guardian System that transforms CCTV cameras into real-time emergency response units.

## Features
- Accident Detection
- Fight Detection
- Automatic Emergency Alerts
- Real-time Video Monitoring

## Technologies Used
- Python
- OpenCV
- YOLOv8
- PyTorch

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run:
   python guardian_prototype.py

## Project Type
Prototype for AI for Social Good Challenge
